# Titanic EDA & Survival Prediction 🚢

## 📊 Project Overview
Analyzed Titanic dataset to understand survival patterns and built a prediction model.

## 🔍 Steps
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Logistic Regression Model

## 📈 Key Insights
- Females had higher survival rate
- 1st class passengers survived more
- Higher fare increased survival chances

## 🛠 Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Scikit-learn

## 🚀 Run Project
pip install -r requirements.txt
python main.py
